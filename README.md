# Quantize

 Quantize is a full-stack arithmetic reasoning engine designed to promote structured mathematical thinking.

## Features

- Dynamic target generation
- Interactive expression builder
- Secure backend expression evaluation 
- Difficulty scaling system
- Constraint validation engine
- Autism-friendly structured design

## Tech Stack

Frontend:
- React.js
- Vite (or Create React App)

Backend:
- Node.js
- Express.js

## Architecture

Quantize/
│
├── client/   # React frontend
├── server/   # Node.js backend

## Core Concepts Implemented

- Tokenization
- Infix to Postfix conversion (Shunting Yard)
- Stack-based expression evaluation
- Constraint validation logic
- Difficulty scaling system

## Setup

### 1. Clone Repository
git clone <repo-url>

### 2. Install Client
cd Client
npm install
npm run dev

### 3. Install Server
cd Server
npm install
node index.js

## Educational Objective

Quantize is built to support structured cognitive development through arithmetic reasoning.
